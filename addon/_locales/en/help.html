Getting Started with
Calibre Search

Thank you for using Calibre Search! If this is the first time using the extension, this short guide is for you.

Prerequisites

  • Ensure Python v3.5 or later is installed and available on PATH.
  • Ensure calibre v5.5 or later is installed and running.

Install the Host Application

The Host Application is necessary for communication between this web browser extension and your calibre library. For non-Windows platforms, see the FAQ.

Windows

  1. Download the latest calibre-search-app-windows.zip from here.
  2. Extract to a temporary folder. e.g. C:\calibre-search-app-windows\
  3. Navigate to C:\calibre-search-app-windows\
  4. Right click on install-host.bat
  5. Select Run as administrator and confirm any prompts.
  6. You can now delete your temporary folder above.
The host application should now be installed.

Configure options

From the configuration page for this extension you have these options below.

Library name

Specify the calibre library name you want to perform your searches against. The default name is Calibre Library but you may have a different folder name.

Replace expressions

List words or regular expressions for text to replace before searching. e.g.

  • by - would change "title by author" into "title author".
  • /\bby\b/gi - same as above but using a regular expression.
  • /\([^\)]*\)/g - would remove anything in parenthesis ().
